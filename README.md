A modern, full-featured e-commerce web application with a sleek dark theme, real-time product search, dynamic cart management, and a complete checkout flow.

## ✨ Features

- **Product Catalog** — Browse products with high-quality images, pricing, and descriptions
- **Category Filtering** — Filter products by Electronics, Fashion, Home & Living, and more
- **Real-Time Search** — Instantly search across the entire product catalog
- **Shopping Cart** — Add, remove, update quantities with live price calculation
- **Checkout Flow** — Complete checkout experience with form validation
- **Responsive Design** — Fully optimized for desktop, tablet, and mobile
- **Dark Theme** — Modern dark glassmorphism UI with smooth animations

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|
| **JavaScript (ES6+)** | Core application logic, DOM manipulation |
| **CSS3** | Custom properties, Flexbox/Grid, animations |
| **HTML5** | Semantic markup, accessibility |
| **LocalStorage** | Cart persistence across sessions |
